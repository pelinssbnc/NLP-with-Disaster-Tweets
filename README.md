# 🌪️ Afet Tweet Sınıflandırma (NLP with Disaster Tweets)

Twitter'daki afet içerikli tweetlerin gerçek mi yoksa
mecazi mi olduğunu NLP ile tespit eden sınıflandırma projesi.

---

## 📌 Genel Bakış

Bu proje **Kaggle NLP Getting Started** yarışmasını temel almaktadır.
Acil durum servisleri ve haber ajansları için tweetlerin
otomatik filtrelenmesine yönelik bir model geliştirilmektedir.

Temel hedefler:
- Metin temizleme ve ön işleme (stopword, lemmatization)
- TF-IDF, Word2Vec veya BERT tabanlı özellik çıkarımı
- F1-Score metriği ile model değerlendirme
